Installation Instructions

Install Node.js

Download and install Node.js from the official website.

Install Python

Download and install Python from the official website.

Install Pip

Ensure you have pip installed. You can download and install it from the official website.

Install Fontra

To install Fontra, run the following command:

pip install fontra

Navigate to Font Folder

Open your terminal and navigate to the font folder using the following command:

cd /workspaces/fornevercollective.github.io/projects/font
